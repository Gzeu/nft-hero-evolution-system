# NFT Hero Evolution System

Sistem complet de evoluție pentru eroi NFT într-un RPG/GameFi pe MultiversX, cu 8 ere succesive, skill trees hibride și economie EGLD integrată.

## Overview
- 8 ere de progresie: Neanderthal → Triburi → Antichitate → Ev Mediu → Industrial → Digital → Cyborg → Superhuman.
- Fiecare Erou are 4 ramuri principale: Fizică, Mentală, Socială, Specială (legată de arhetip Tarot/Planetă).
- Skill tree per erou: 20 noduri lifetime, max 25 puncte talent alocate.
- Evoluția pe ere face upgrade semantic talentelor (ex: Stone Fury → Rifle Fury) și deblochează noi noduri.
- Economie GameFi cu EGLD, NFT-uri tematice și integrare DAO/Guild.

Detaliile complete pentru Arhetipurile Marte și Mercur sunt în `docs/archetypes/`.
