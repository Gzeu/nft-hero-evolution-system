# Arhetip: Marte (Războinic Ofensiv)

**Focus**: Agresiv, damage burst/martial fury. Ramură Specială = Fury bursts.

## Skill Tree Text
```
Tier 1 (Liniare Core - Toate Ramurile)
├── Fizică: Brutal Strike → Ramifică: Tank | Berserker | Hunter
├── Mentală: Battle Rage → Strateg | Predict | Overclock
├── Socială: War Cry → Rally | Intimidate | Conquer
└── Specială: Mars Fury → Burst | Sustain | AoE

Tier 2+ (Ramificări per Eră Noi Noduri)
Fizică ── Tank (Def) ── Berserker (Dmg) ── Hunter (Single)
Mentală ─ Strateg (Buff) ─ Predict (Debuff) ─ Overclock (Self)
Socială ─ Rally (Ally) ─ Intimidate (Fear) ─ Conquer (Loot)
Specială ─ Burst (Nuke) ─ Sustain (Lifesteal) ─ AoE (Clear)
```
